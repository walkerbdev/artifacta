# ML Pipeline Example

ResNet50 training pipeline for CIFAR-10 image classification.

## Structure

- `src/` - Source code
  - `model.py` - Model architecture
  - `train.py` - Training script
  - `data.py` - Data loading
- `utils/` - Utility functions
  - `metrics.py` - Evaluation metrics
- `config/` - Configuration files
  - `train_config.yaml` - Training hyperparameters

## Usage

```bash
python src/train.py
```

This example demonstrates logging a complete multi-file project as a single artifact.
